Project Title: Photo Searching Web App with Google OAuth and Firebase Integration


Overview:

In this project, I developed a photo searching web application that seamlessly integrates the Pinterest design aesthetic. Users can search for and bookmark their favorite photos, with an interface inspired by the popular visual discovery platform. Leveraging the Pixabay API, users can easily find images based on keywords, which are dynamically displayed in the browser. Additionally, users can log in and out using Google OAuth for enhanced security and convenience. The application also enables users to store their favorite photos using Firebase and Firestore, ensuring their data is securely saved in the cloud.


Key Features:

  1. Pixabay API Integration: Utilized the Pixabay API to fetch photos based on user search queries, providing a vast selection of images for users to explore.
  2. Google OAuth Authentication: Implemented Google OAuth for seamless and secure user authentication, allowing users to log in and out using their Google accounts.
  3. Firebase and Firestore Integration: Integrated Firebase and Firestore to enable users to store their favorite photos in the cloud, providing a personalized experience across devices.
  4. Bookmarking Functionality: Enabled users to bookmark their favorite photos with a simple click, allowing them to curate their own collections effortlessly, with a design reminiscent of Pinterest's bookmarking feature.
  5. Tailwind CSS Styling with Pinterest Design: Styled the application using Tailwind CSS, incorporating elements and design principles inspired by Pinterest for a visually appealing and engaging user experience.

Technologies Used:

  • Frontend: HTML5, Tailwind css, Next js
  • Database: Firebase Firestore
  • Authentication: Google OAuth
  • Authentication Management: NextAuth
